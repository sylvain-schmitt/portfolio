Explorer

Le projet Explorer est comme son nom l'indique un explorateur de fichier écrit en php.

Il est possible d'effectuer des opérations de création, suppréssion de dossier ainsi que de fichier mais seul les utilisateurs ayant un rôle Admistrateur peuvent accéder à ces fonctionnalitées.

Enfin pour tous les utilisateurs connecter il est possible de visualiser les fichiers et de les télécharger au format pdf.

Date: Novembre 2020

Technologies utilisée: PHP/HTML/CSS
